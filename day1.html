    Ques5: Difference between block level and inline element.
    Ans: Block level:- 1. Block level element always begin in a next line and it extends from left to right
                       2. It takes the full width available on the screen
                       3. Browser automatically adds some space before and after the element .
                            eg:- table, div , h1, ol, ul etc
        Inline: Does not begin on new line. It only takes the space that is necesary for it.
                            eg: span, img, a, input, label
